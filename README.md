# my_python_note

这是一个「刚毕业求职 Demo 网页」示例，适合零基础先看到成果再慢慢学代码。

## 你要的“直接地址”

你现在就可以访问这个地址（本机）：

- <http://localhost:8000>
- <http://127.0.0.1:8000>

> 先说明：我在这个对话里不能给你一个长期公网网址（比如别人也能直接打开的线上域名），
> 但可以给你 **本地可用地址 localhost**，你运行 1 条命令就能马上看到页面。

---

## 零基础：复制这两步就行

### 第 1 步：启动网页

在项目目录打开终端后运行：

```bash
python3 -m http.server 8000
```

> Windows 也可以用：`python -m http.server 8000`

### 第 2 步：打开浏览器

访问：<http://localhost:8000>

看到页面就说明成功了 ✅

停止服务：在终端按 `Ctrl + C`。

---

## 如果你想让同一 Wi-Fi 的手机也能看

启动时改成：

```bash
python3 -m http.server 8000 --bind 0.0.0.0
```

然后在电脑终端查你的局域网 IP（例如 `192.168.1.23`），手机访问：

- `http://你的局域网IP:8000`
- 例子：`http://192.168.1.23:8000`

## 你问“为什么 GitHub 还没更新”

因为之前只是把代码提交到了**本地 Git**，还需要再执行一次 **push** 才会上 GitHub。

按顺序执行（在项目目录）：

```bash
git remote -v
git branch --show-current
git push -u origin 当前分支名
```

如果你不知道“当前分支名”是什么，可以直接用：

```bash
git push -u origin $(git branch --show-current)
```

如果 push 要你登录：
- 用 GitHub Desktop 登录后再 push，或者
- 用命令行配置 token / SSH key。
